Acesta este un proiect Django care simuleaza un magazin online de haine.  
Aplicatia permite gestionarea produselor, afisarea lor pe site, navigare prin categorii, sistem de utilizatori si panou de administrare complet prin Django Admin.

Proiectul este realizat ca aplicatie universitara si demonstreaza concepte de backend, routing, template rendering, lucrul cu modele si baze de date in Django.
